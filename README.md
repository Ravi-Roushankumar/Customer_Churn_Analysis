# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data using Python. 
The goal is to understand why customers leave a service and identify important patterns in the dataset.

## Dataset
The dataset used in this project is "Customer Churn.csv".

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project File
TCA.ipynb contains the complete data analysis process including:
- Data loading
- Data cleaning
- Exploratory Data Analysis (EDA)
- Visualization

## Objective
To analyze customer churn behavior and understand the factors that influence customer retention.
